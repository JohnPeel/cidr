Aggregates a list of included and excluded CIDRs.

    cidr [OPTIONS] <CIDR>...

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -e, --exclude <CIDR>...    CIDRs to exclude from the list

ARGS:
    <CIDR>...    CIDRs to include in the list
